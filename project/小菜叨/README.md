### 项目简介

**小菜叨**是一款包含厨友交流、美食做法经验分享、菜谱搜索、厨友交流功能的app。

创意：

* 第三方数据接口与jFinal服务器的结合，基于第三方接口提供的大量数据以及jFinal服务器提供的用户交互数据，提供更为详尽，有效的数据。
* 论坛式交流，论坛式的发帖和评论体系与交流方式，提供给大家展示的天地。
