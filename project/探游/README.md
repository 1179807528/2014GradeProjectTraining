### 项目简介

**校脸**APP是一款高校间线上线下旅游交友软件。

用户可通过手机号与验证码注册，登陆后可进行的操作有：

* 修改个人资料
* 发送出游邀请并按条件匹配导游
* 接受或拒绝他人出游邀请
* 与他人实时聊天
* 查看地图景点覆盖物
* 记录出游路径
* 查看历史出游路径
* 发表对本次出游高校及导游的评价
* 查看他人对某高校或用户的评价
* 发表评价后刮刮乐抽奖
* 通过他人对自己的评价获得积分并兑换奖品
